# Updated Deployment Folder

This folder contains revised deployment notes and scripts for the compact JINT repository.

The scripts are designed as reproducibility and smoke-test helpers for:
- Main Table 2 / Main Fig. 3: INT8 CPU-only inference,
- Main Fig. 5 / Main Table 5: 30-minute Raspberry Pi 5 thermal and scheduling evidence,
- Supplementary Note S2: queue-thermal scheduling interpretation.

Live UAV and Raspberry Pi 5 stress collection must still be performed on the physical hardware. The public repository scripts validate released logs and provide reproducible entry points.
