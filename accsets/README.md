This folder stores redistributable manuscript and supplementary figure assets for the JINT compact submission.

The main manuscript contains Fig. 1-Fig. 6, and the supplementary material contains Fig. S1-Fig. S5. When final high-resolution figure files are redistributable, they are placed in the corresponding subfolders.

Raw numerical data used to regenerate plotted figures are stored under `results/figures_raw/`, while scripts for reproducing plots are stored under `scripts/`.

Some schematic figures may be provided only as final rendered images if they were manually drawn or assembled from anonymized visual materials.
